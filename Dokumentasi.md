## Clone Github & Menjalankan aplikasi
---

### Clone Github

git clone https://github.com/USERNAME/responsi-infra.git 
cd responsi-infra 

![Clone](<Screenshot 2026-06-08 104604.png>)
---

### Menjalankan Aplikasi
---

Masuk ke folder project lalu jalankan: 
docker compose up -d 
![Menjalankan aplikasi](<Screenshot 2026-06-08 111820.png>).

Untuk melihat status container: 
docker ps 
![Status](<Screenshot 2026-06-08 112230.png>).

Untuk melihat log: 
docker compose logs 

![log1](<Screenshot 2026-06-08 120028.png>) 

![log2](<Screenshot 2026-06-08 120038.png>)
